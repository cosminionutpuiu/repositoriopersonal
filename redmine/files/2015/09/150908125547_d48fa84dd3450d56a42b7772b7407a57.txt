- En primer lugar se comprobó la existencia del fichero principal de configuración que se encuentra en la siguiente ruta:
	
	var/lib/tomcat6/webapps/bigbluebutton/WEB-INF/classes/bigbluebutton.properties
	
	cd var/lib/tomcat6/webapps/
	ll

- Al comprobar que dicho fichero no existía, se procedió a la instalación de la aplicación siguiendo los pasos básicos de instalación de la página oficial:
	
1. Se verifica la versión instalada de ruby.
	ruby -v

2. Se verifica la versión instalada de gem.
	gem -v

3. Se instalan los repositorios.
	wget http://ubuntu.bigbluebutton.org/bigbluebutton.asc -O- | sudo apt-key add -
	echo "deb http://ubuntu.bigbluebutton.org/lucid_dev_081/ bigbluebutton-lucid main" | sudo tee /etc/apt/sources.list.d/bigbluebutton.list

4. Se verifican las fuentes.
	grep "lucid multiverse" /etc/apt/sources.list

5. Se actualiza ubuntu.
	sudo apt-get update
	sudo apt-get dist-upgrade

6. Se instala Libre Office.
	wget http://bigbluebutton.googlecode.com/files/openoffice.org_1.0.4_all.deb
	sudo dpkg -i openoffice.org_1.0.4_all.deb
	
	sudo apt-get install python-software-properties

	sudo apt-add-repository ppa:libreoffice/libreoffice-4-0
	sudo apt-get update

	sudo apt-get install libreoffice-common
	sudo apt-get install libreoffice

7. Se comprueba la versión instalada de ffmpeg
	ffmpeg -version
	
8. Se instala BigBlueButton
	sudo apt-get install bigbluebutton
	
9. Se instala API Demos
	sudo apt-get install bbb-demo

10. Se configuran todos los archivos con la correcta IP
	sudo bbb-conf --setip <ip_or_hostname>
	
11. Se reinicia BigBlueButton
	sudo bbb-conf --clean
	sudo bbb-conf --check
	
12. Para la configuración en Red Profesional se eliminan las API Demos
	sudo apt-get purge bbb-demo
	
13. Se pide la seguridad salt
	sudo bbb-conf --salt
